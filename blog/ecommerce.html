Building an E-commerce Website with Angular

Back to Portfolio

Introduction

In today’s digital age, e-commerce has become a vital part of business. In this blog, I will guide you through the process of creating a fully functional e-commerce website using Angular, one of the most popular frameworks for building dynamic web applications.

Step 1: Setting Up the Angular Project

First, install Angular CLI if you haven’t already. Then, create a new Angular project:

ng new ecommerce-website

This command will create a new Angular project named ecommerce-website with all the necessary files and dependencies.

Step 2: Creating Components

Next, generate the necessary components for your application. The key components you’ll need include:

  • Product List: Displays all available products.
  • Product Details: Shows detailed information about a selected product.
  • Shopping Cart: Manages the items that users want to purchase.

Run the following commands to create these components:

ng generate component product-list
ng generate component product-details
ng generate component shopping-cart

Step 3: Setting Up Routing

Angular’s routing allows users to navigate between different components. To set up routing, modify your app-routing.module.ts file:

import { NgModule } from '@angular/core'; import { RouterModule, Routes } from '@angular/router'; import { ProductListComponent } from './product-list/product-list.component'; import { ProductDetailsComponent } from './product-details/product-details.component'; import { ShoppingCartComponent } from './shopping-cart/shopping-cart.component'; const routes: Routes = [ { path: '', component: ProductListComponent }, { path: 'product/:id', component: ProductDetailsComponent }, { path: 'cart', component: ShoppingCartComponent }, ]; @NgModule({ imports: [RouterModule.forRoot(routes)], exports: [RouterModule] }) export class AppRoutingModule { }

This configuration allows you to navigate to the product details and shopping cart using routes.

Step 4: Building the UI

Utilize Bootstrap for styling your components. Implement a responsive design that provides an excellent user experience. For example, you can use Bootstrap classes to create a grid layout for your product listings: 

<div class="row"> <div class="col-md-4"> <div class="card"> <img src="product-image.jpg" class="card-img-top" alt="Product"> <div class="card-body"> <h5 class="card-title">Product Name</h5> <p class="card-text">Product Description</p> <a href="/product/1" class="btn btn-primary">View Details</a> </div> </div> </div> </div>

Step 5: Managing State with Services

To manage the state of your application, such as the shopping cart items, create a service. This service will handle adding, removing, and retrieving items:

import { Injectable } from '@angular/core'; @Injectable({ providedIn: 'root', }) export class CartService { private items = []; addToCart(product) { this.items.push(product); } getItems() { return this.items; } clearCart() { this.items = []; return this.items; } }

Step 6: Testing the Application

Once you have implemented the basic features, it’s crucial to test your application. Use Angular's built-in testing utilities to create unit tests for your components and services.

Conclusion

By following these steps, you can create a powerful e-commerce platform using Angular. This project not only showcases your development skills but also your understanding of modern web technologies. Continuous enhancements can be made, such as integrating payment gateways, user authentication, and inventory management.
